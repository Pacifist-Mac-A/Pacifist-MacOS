## Repairing a Broken Installation

One of the most useful capabilities of <a href="#">Pacifist Mac</a> is diagnosing and fixing damaged
installations:

### Verifying Files Against the Package

Point Pacifist at an installer package and it compares the files on your system to the originals,
identifying anything missing, altered, or corrupted. This verification pinpoints the source of
problems that a full reinstall would address blindly.

### Reinstalling Only What Is Broken

Once the damaged files are identified, the <a href="#">Pacifist installer</a> utility reinstalls
exactly those files from the original package. This targeted repair fixes the issue without the time
and disruption of reinstalling an entire application or system component.
